- **Modern UI** - Clean, responsive interface with REAYRT branding

## Technology Stack

- HTML5 / CSS3 / JavaScript
- Hosted on GitHub Pages
- Integrates with [REAYRT Core API](https://api.reayrt.com)

## Pages

- **Landing Page** (`index.html`) - Home page with API status indicator
- **Mesh Editor** (`editor.html`) - Interactive mesh editor interface

## Local Development

Simply open `index.html` in a web browser to run locally.

## API Integration

The application connects to the REAYRT Core API at `https://api.reayrt.com` for backend functionality and health monitoring.
